## 快速开始

### 1. 创建所有默认活动

```bash
./run_3001_all_activities.sh
```

默认会创建：优惠券、每日签到、救援金、每日任务、周卡月卡、新版代理、工单系统

### 2. 创建指定活动

```bash
# 只创建救援金
./run_3001_all_activities.sh rescue

# 创建多个活动（逗号分隔）
./run_3001_all_activities.sh coupon,signin,rescue,dailyTasks

# 创建所有活动
./run_3001_all_activities.sh coupon,signin,redRain,champion,luckyDoubleBonus,giftCodes,megaJackpot,activityGuide,banner,codeWashing,customizePopup,dailyTasks,giftPack,inmail,inviteTurntable,loginPopup,newagent,newagentRank,order,ranking,rechargeGiftPack,rechargeWheel,rescue,tag,weekCard,withdrawalTimeout
```

### 3. 自定义语言配置

```bash
# 只使用中文和英语
LANGUAGES=zh,en ./run_3001_all_activities.sh rescue

# 使用所有三种语言（默认）
LANGUAGES=zh,en,hi ./run_3001_all_activities.sh rescue
```

## 常见问题

### 1. 如何查看支持的活动类型？

运行脚本时会自动显示所有支持的活动类型。

### 2. 活动创建失败怎么办？

检查日志输出，常见原因：
- 管理员登录失败：检查账号密码
- 网络连接问题：检查域名是否可访问
- 活动配置错误：检查活动参数是否正确

### 3. 如何验证活动是否创建成功？

1. 登录后台管理系统：https://ar666999.club
2. 进入对应的活动管理页面
3. 检查是否有新创建的活动记录

### 4. 如何添加新的语言支持？

1. 在 `k6/config/tenantLanguageConfig.js` 中添加语言配置
2. 在各个活动的创建函数中添加对应语言的翻译
3. 运行脚本时指定新的语言代码

## 注意事项

1. **活动创建顺序**: 脚本会按照指定的顺序依次创建活动
2. **活动间隔**: 每个活动创建之间会间隔3秒，避免请求过快
3. **超时设置**: 整个脚本最长执行时间为600秒（10分钟）
4. **错误处理**: 某个活动创建失败不会影响其他活动的创建
5. **语言一致性**: 确保所有活动使用相同的语言配置，保持用户体验一致
